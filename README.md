## 📖 About This Repository

This repository contains coursework, experiments, and a hands-on AI project from the **Artificial Intelligence** course at UGM. The course covers the theoretical foundations of AI through to practical machine learning implementation using Python and Google Colab.


## 🧠 Topics Covered

**Foundations of AI**
- History and overview of Artificial Intelligence
- Types of AI: Narrow AI, General AI, Machine Learning, Deep Learning
- Problem-solving and search algorithms

**Machine Learning**
- Supervised learning (classification, regression)
- Unsupervised learning (clustering)
- Model evaluation (accuracy, precision, recall, F1-score)
- Overfitting, underfitting, and train-test split

**Practical Implementation**
- Data preprocessing and feature engineering
- ML model building with Scikit-learn
- Hands-on project: AI model development (see below)

## 🚀 How to Run

All notebooks are designed to run on **Google Colab** — no local setup needed.

1. Open the `.ipynb` file in this repo
2. Click **"Open in Colab"** button
3. Run all cells (`Runtime → Run all`)

Or run locally:

```bash
# Clone the repository
git clone https://github.com/insyania/AI-SM4.git
cd AI-SM4

# Install dependencies
pip install numpy pandas scikit-learn matplotlib seaborn

# Open notebook
jupyter notebook
```
